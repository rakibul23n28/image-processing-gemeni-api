# AI-Based Scene Narrator for Blind Users 👁️‍🗨️🎙️

This project uses a live camera feed and Google's Gemini AI to describe scenes in natural language, tailored for blind or visually impaired users. It supports voice commands, multiple languages, and audio playback of scene descriptions.

## 🌟 Features

- 🎥 **Live Camera Feed** using OpenCV
- 📸 **Capture Images** via button or voice command ("capture")
- 🧠 **Scene Description** via Google Gemini AI
- 🔊 **Text-to-Speech (TTS)** output in selected language
- 🌐 **Multilingual Support**: English, Bangla, Spanish, French, German, Hindi
- 🎤 **Voice Command Mode** using SpeechRecognition
- 💾 **Automatic saving** of captured images and AI responses
